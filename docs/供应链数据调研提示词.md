# 供应链数据深度调研提示词

你可以将此提示词发给 AI 助手（如 Claude、ChatGPT、Kimi 等）来获取上市公司的供应链关系、股东和质押数据。

---

## 任务

帮我调研以下 **5 家中国 A 股上市公司**的最新供应链关系、前十大股东和股权质押情况。数据将用于构建企业知识图谱和供应链传导风险模型（TRACE 项目）。

请严格按 JSON 格式输出，以便程序直接解析。

## 公司列表

1. **宁德时代 (300750.SZ)** — 动力电池
2. **比亚迪 (002594.SZ)** — 新能源车+电池
3. **立讯精密 (002475.SZ)** — 消费电子/苹果供应链
4. **隆基绿能 (601012.SH)** — 光伏
5. **贵州茅台 (600519.SH)** — 白酒

## 需要的数据

### 1. 前5大客户（按营收占比）

对每家公司，列出前5大客户：

```json
{
  "customers": [
    {
      "company_code": "300750.SZ",
      "customer_name": "特斯拉 (Tesla)",
      "revenue_pct": 18.5,
      "source": "2024年年报",
      "note": "连续多年第一大客户"
    }
  ]
}
```

要求：
- `revenue_pct` 为营收占比百分比（如 18.5 表示 18.5%）
- `source` 标注数据来源
- 如果年报未披露具体名称（仅写"客户一"），请注明并尽量推测

### 2. 前5大供应商（按采购占比）

对每家公司，列出前5大供应商：

```json
{
  "suppliers": [
    {
      "company_code": "300750.SZ",
      "supplier_name": "天赐材料 (002709.SZ)",
      "procurement_pct": 8.3,
      "source": "2024年年报",
      "note": "电解液供应商"
    }
  ]
}
```

要求：
- 如果供应商是上市公司，标注股票代码
- `procurement_pct` 为采购占比百分比

### 3. 前十大股东

对每家公司，列出前十大股东（取最新报告期，通常为 2024Q4 或 2025Q1）：

```json
{
  "shareholders": [
    {
      "company_code": "300750.SZ",
      "holder_name": "香港中央结算(代理人)有限公司",
      "hold_amount_shares": 4200000000,
      "hold_ratio_pct": 22.50,
      "holder_type": "境外法人",
      "report_date": "2024-12-31"
    }
  ]
}
```

要求：
- `hold_ratio_pct` 为持股比例百分比
- `holder_type` 取：境内自然人/境外法人/国有法人/投资公司/开放式投资基金/全国社保基金/券商/其他
- 标注报告期

### 4. 股权质押统计

对每家公司，提供最新质押数据：

```json
{
  "pledge": [
    {
      "company_code": "300750.SZ",
      "pledge_ratio_pct": 0.0,
      "pledge_count": 0,
      "total_share": 18682400000,
      "update_date": "2026-05-26"
    }
  ]
}
```

要求：
- `pledge_ratio_pct` 为质押占总股本比例百分比
- 如果无质押，`pledge_ratio_pct` 填 0

## 输出格式

请将所有结果合并为一个 JSON 文件，结构如下：

```json
{
  "meta": {
    "project": "TRACE",
    "generated_date": "YYYY-MM-DD",
    "data_sources": ["公司年报", "东方财富", "同花顺", "Tushare"],
    "disclaimer": "部分数据来自公开披露，占比为近似值"
  },
  "companies": {
    "300750.SZ": {
      "name": "宁德时代",
      "customers": [...],
      "suppliers": [...],
      "shareholders": [...],
      "pledge": [...]
    },
    ...
  }
}
```

## 数据来源建议

1. **东方财富网** (eastmoney.com) — 搜索股票代码 → "股东研究" → "前十大股东"
2. **巨潮资讯网** (cninfo.com.cn) — 搜索公司最新年报 PDF
3. **同花顺** (10jqka.com.cn) — F10 → 股东研究
4. **Tushare Pro** (tushare.pro) — top10_holders / pledge_stat 接口（需积分）
5. **公司年报** — 第三节"管理层讨论与分析"中"主营业务分析"章节

## 注意事项

- 部分公司年报可能不披露具体客户/供应商名称（以"客户一""供应商一"代称），请如实标注
- 持股数为近似值即可，精确到百万股
- 数据时效性以最新可获取的报告为准
- 如果某个数据源无法获取，尝试使用替代来源并在 note 中说明
