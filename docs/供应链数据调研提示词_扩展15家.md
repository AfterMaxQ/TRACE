## 每家公司需要的四类数据

### 1. 前5大客户
```json
{"customer_name": "客户名称", "revenue_pct": "占比(数字或描述)", "source": "2025年报 第三节"}
```
注意：年报通常只披露合计占比而非逐一列出；若未披露具体名称，标注"未披露"并填写合计占比。

### 2. 前5大供应商
```json
{"supplier_name": "供应商名称", "procurement_pct": "占比", "source": "2025年报 第三节"}
```
注意：若该供应商为上市公司，尽量标注股票代码。

### 3. 前十大股东（2025年报/2026Q1最新）
```json
{"holder_name": "股东名称", "hold_ratio_pct": 数值, "holder_type": "境内自然人/境外法人/国有法人/投资公司/基金/券商/其他", "report_date": "2024-12-31"}
```

### 4. 股权质押统计（中登公司/东方财富最新数据）
```json
{"pledge_ratio_pct": 数值, "pledge_count": 数量, "total_share": 总股本, "update_date": "YYYY-MM-DD", "source": "东方财富/中登公司"}
```

```json
    }
  }
}
```

## 数据来源

1. 东方财富网 → 股票代码 → F10 → 股东研究 / 经营分析
2. 同花顺 → 股票代码 → F10 → 财务分析 → 主要客户/供应商
3. 巨潮资讯网 → 搜索公司最新年报 PDF
4. 金融界 → 股票代码 → 质押数据（中登公司）

## 注意事项

- 半导体/医药行业公司通常不披露具体客户/供应商名称（商业机密），如实标注
- 国有控股公司（如中芯国际、海螺水泥）注意区分国有法人和市场化股东
- 部分公司客户/供应商为集团内部关联方，需标注
